WP Component enable modular macro-template for created beautiful page with custom fields

== Description ==

Create powerful, intelligent and reusable macro templates.
Easily define a customized content structure.

All components that you create are reusable as many times as you like, with any content you can imagine.
When you create a component, itâ€™s easy to specify the exact content that you need.
You define your own component, your own content and your own containers.

Then add any chapter you like (the component) in any format you choose, with any kind of content, wherever you like, as many times as you like, using your WordPress content manager!

Then add this beautiful the_story() wherever you like, within a page, in an existing template, or in a custom template.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wpcomponent` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php the_story(); ?>` in your templates
4. Create Component in "wpcomponent" folder in your theme
5. View [WP component Website](http://wpcomponent.com/docs "WPComponenet for WordPress") for full documentation
